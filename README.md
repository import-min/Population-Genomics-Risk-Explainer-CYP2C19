# Population Genomics Risk Explainer

Genetic data is often talked about as if it can predict someone’s health or future. In reality, most genetic information comes from studying large groups of people, not individuals.

This project uses simple visualizations and examples from real population genetics data to show how common certain genetic variants are in different populations, and why that information is easy to misunderstand. The focus is on explaining what genetic “risk” actually means, what it does not mean, and how population averages can be misleading when applied to a single person.

Current examples focus on clinically relevant pharmacogenomic variants from gnomAD (v4.1.0), such as CYP2C19, to ground explanations in real-world data.

Rather than predicting disease, this project is meant to help students understand how genetic data is used, how it can be misinterpreted, and why careful interpretation matters in medicine and public health. The materials and visualizations in this project are designed to be adaptable for STEM outreach programs, including workshops developed through the Marcus Harris Foundation that walk students through real biomedical data step by step, starting from basic concepts.
